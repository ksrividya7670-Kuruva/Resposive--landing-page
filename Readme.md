# Tailwind CSS Landing Page

## Project Overview

This project is a responsive landing page built using **HTML** and **Tailwind CSS (CLI)**. The page is designed to showcase a modern product website with a navigation bar, hero section, feature cards, call-to-action buttons, and a floating chat button.

## Features

* Responsive navigation bar
* Modern hero section with gradient background
* Call-to-action buttons
* Feature cards (Tasks, Project Status, Team Activity)
* Floating chat button
* Clean and responsive design
* Built using Tailwind CSS utility classes

## Technologies Used

* HTML5
* Tailwind CSS (CLI)

## Installation

1. Clone or download the project.
2. Install dependencies.

```bash
npm install
```

3. Start the Tailwind CSS watcher.

```bash
npx @tailwindcss/cli -i ./src/input.css -o ./output.css --watch
```

4. Open `index.html` in your browser.
